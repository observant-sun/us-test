# Тестовое задание

Приложение на Spring. Реализован рест `GET /excel-file`, принимающий два обязательных параметра:
  - `pathToFile` - string, путь к локальному файлу XLSX,
  - `n` - integer, какое по порядку минимальное число (начиная с 1) нужно вернуть из файла.

Приложение прочитывает первую колонку из Excel-файла, останавливаясь на первой пустой ячейке. Если в ячейке приложение найдет нечисловые данные, выбрасывается исключение.

## Сборка и запуск

### Запуск для разработки
Сборку и запуск приложения из папки проекта при разработке можно делать командой:
```
./gradlew bootRun
```

### Сборка и запуск JAR
Собрать запускаемый JAR-файл можно командой:
```
./gradlew bootJar
```
В папке build/libs появится JAR-файл. Его можно запустить, выполнив из той папки:
```
java -jar us-test-0.0.1-SNAPSHOT.jar
```
